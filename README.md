# INO-Arduino-Obstacle-Avoiding
Robot Obstacle Avoiding menggunakan Mikorkontroler Arduino Uno. <br>
**Catu daya menggunakan  Baterai Lipo 2S (7,4v)** 

## Bahan-bahan Utama
- Arduino Uno
- Sensor Ultrasonic HC-SR04
- Motor Driver L298N
- Servo Tower Pro Sg90
